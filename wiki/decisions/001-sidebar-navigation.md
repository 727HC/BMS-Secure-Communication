# ADR-001: 사이드바 네비게이션 전환

## 상태
Accepted

## 맥락
기존 상단 네비게이션(탑바 + 섹션 칩 + 운영 맥락 패널 + 페이지 탭)이 수직 공간을 과도하게 차지. EV Data Interface 레퍼런스 기반 리디자인 시 콘텐츠 영역 부족.

## 결정
상단 네비게이션을 64px 좌측 아이콘 사이드바로 전환.
- 섹션 칩, 운영 맥락 패널, 페이지 탭 제거
- 상단바는 52px 높이로 축소 (페이지 제목 + 유저 정보만)
- 모바일: 사이드바 숨김 + 햄버거 메뉴

## 결과
- 수직 콘텐츠 공간 ~200px 확보
- 전체 페이지(여권 목록, 상세 등)에 동일 적용
- 모바일 대응 필요 (lg:hidden / hamburger)
