# BATP 수동 QA 체크리스트

## 1. 공통 진입
- [ ] `http://127.0.0.1:3001/` 접속 시 대문이 정상 노출된다.
- [ ] 로그인/계정 등록 전환이 자연스럽다.
- [ ] 로그인 후 사이드바/상단바가 깨지지 않는다.

## 2. 대시보드
- [ ] KPI 카드 3개가 깨지지 않는다.
- [ ] 화학계열/상태 분포 카드가 같은 톤으로 읽힌다.
- [ ] 테이블 액션(`다음 조치`, `상세`, `⋯`)이 동작한다.

## 3. 배터리 여권 등록부
- [ ] 검색/상태 필터/정렬이 동작한다.
- [ ] 제조사 계정에서 `여권 발급` 모달이 열린다.
- [ ] 3단계 발급 wizard 완료 후 목록에 새 여권이 보인다.

## 4. 여권 상세
- [ ] 상단 hero, 핵심 식별, 운영 상태 카드 가독성이 자연스럽다.
- [ ] 탭(`개요/규제·소재/운영 이력/진단 데이터/증빙`) 전환이 즉시 반응한다.
- [ ] action rail 드래그/축소/확장이 된다.
- [ ] 상세 모달(바인딩/정정/원자재 연결 등)이 정상 열린다.

## 5. 원자재
- [ ] 원자재 검색이 된다.
- [ ] 제조사 계정에서 `원자재 등록` 모달이 열린다.
- [ ] 등록 후 목록과 상세 모달이 정상 갱신된다.

## 6. 정비 운영
- [ ] EV 제조사에서 정비 요청이 가능하다.
- [ ] 서비스 계정에서 정비 완료 기록이 가능하다.
- [ ] 사고 기록 등록 모달이 정상 열린다.

## 7. 회수 운영
- [ ] EV 제조사에서 분석 요청 버튼이 보인다.
- [ ] 규제/서비스 계정에서 재활용 판정 흐름이 보인다.
- [ ] 추출/폐기 모달이 정상 열린다.

## 8. BMU 데이터
- [ ] 여권 ID 입력 후 조회가 된다.
- [ ] 자동 새로고침 토글이 동작한다.
- [ ] 판독 기록 표와 상태 badge가 정상이다.

## 9. QR / NFC
- [ ] 수동 입력 조회가 된다.
- [ ] 카메라 스캔 진입이 자연스럽다.
- [ ] NFC 미지원 환경에서 안내 패널이 보인다.

## 10. 감사 로그
- [ ] action 필터와 쓰기 작업만 체크박스가 동작한다.
- [ ] 각 로그 row 확장이 자연스럽다.
- [ ] 페이지네이션이 정상이다.

## 11. 마감 확인
- [ ] 한글 문구가 어색하지 않다.
- [ ] 영어 라벨이 의도치 않게 남아있지 않다.
- [ ] 모바일 폭에서 주요 화면이 깨지지 않는다.
