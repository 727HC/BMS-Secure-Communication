# passport/activity-log/

Passport 공통 활동 로그의 날짜별 파일을 모아두는 하위 허브다.

## 이 폴더에 두는 것
- 날짜별 공통 작업 기록
- 협업 세션 요약과 작업 주체 정보
- 코드/문서/운영 변경의 상위 타임라인 요약

## 기록 규칙
- 새 공통 기록은 해당 날짜 파일에 추가한다.
- 같은 날짜 안에서는 `Session N` 형식으로 이어 붙인다.
- 세부 시행착오나 agent 전용 상세 로그는 `_archive/` 쪽으로 보낸다.
