## 교훈

- 같은 prop을 여러 곳에 전달하면서도 형식만 다른 라벨/플래그(예: `org === 'ManufacturerMSP'` 검사가 4–5곳에 산재)는 1줄짜리 어댑터 훅(`useOrgRoles`)으로 합치면 페이지 import / inline 분기 모두 줄어든다. 8줄짜리 훅이 5개 페이지에서 동일 패턴을 통합한다.
- 도메인별 권한 조합 (`canRequestAnalysis = isEVManufacturer`)은 공용 훅에 넣지 말고 도메인 폴더 안의 권한 훅에서 `useOrgRoles`를 컴포지션해서 만든다. 공용 훅이 도메인 노이즈로 더러워지는 걸 막을 수 있다.
- 페이지 한 줄짜리 라벨 패턴이라도 두세 개 페이지에서 비슷한 모양으로 반복되면 별도 훅으로 빼는 편이 미래의 변경 폭을 줄인다.
