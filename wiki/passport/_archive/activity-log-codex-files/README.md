# passport/_archive/activity-log-codex-files/

Codex 전용 상세 로그의 날짜별 파일을 모아두는 하위 허브다.

## 이 폴더에 두는 것
- Codex가 수행한 상세 변경 기록
- 검증 명령, 시행착오, 세부 정리 메모
- 공통 활동 로그에 전부 올리기엔 너무 긴 보조 타임라인

## 기록 규칙
- 새 Codex 상세 기록은 해당 날짜 파일에 추가한다.
- 공통 타임라인 반영이 필요하면 `passport/activity-log/` 쪽 날짜 파일에도 요약을 남긴다.
- archive 트랙은 세부 추적용이므로, 공식 요약은 공통 활동 로그를 우선 본다.
