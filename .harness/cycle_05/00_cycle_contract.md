# Cycle 05 — Complete Teardown & Rebuild

This is cycle 5, not the whole project.
This cycle contains 15 micro-loops.
The project is not complete until 10 to 15 full cycles are done.

## Cycle Objective
Complete visual teardown and rebuild. Delete decorative elements, use design system classes, create real hierarchy.

## Hypothesis
"Clean Product" — Delete all decoration that doesn't carry information. Use the existing design system. Create 4-level font hierarchy (32/20/14/12). Centered login, stat-card dashboard, border-radius on interactive elements.

## Key Actions
1. Login: centered card, delete hero, delete footer tags, simple org selector
2. Dashboard: stat cards + charts + 3-4 recent items, no shortcuts section
3. All pages: replace inline styles with design system classes
4. Delete stamps/seals, use badge pills
5. Add border-radius to inputs/buttons/cards
6. Real font hierarchy: 32px title, 20px section, 14px body, 12px caption
